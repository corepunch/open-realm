// Architecture

Built for Hackers.

Pure C core with a Lua scripting layer. No game engine abstractions — just SDL2, OpenGL, and direct asset format parsers. Designed to be read, forked, and understood.


The renderer supports multiple file format families side-by-side: MDX for Warcraft III, M2/WMO for WoW, M3 for StarCraft II. Each shares a common OpenGL render pipeline and scene graph.

r_m2.c — M2 character renderer 
// Determine visible geosets from outfit
void M2_CharacterGeosetVisible(
    struct m2_model_t *model,
    struct outfit_t *outfit
) {
    for (int i = 0; i < model->num_geosets; i++) {
        struct m2_geoset_t *gs = 
            &model->geosets[i];
        gs->visible = outfit_has_geoset(
            outfit, gs->id
        );
    }
}
